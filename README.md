# 🎰 Rock Paper Scissors
This is my mini project in Python. A rock, paper, scissors game, who doesn't know about this? Sure, we do. Straight to the point, the game is still the same, but your opponent in this game is a bot. Not AI, it's actually the bot where I code it to play this game.

### 🕹️ How does the script work?
So basically, you just run the code, and you get a message if you want to play the game or not. By inputting 'y ', you'll start the game, and you pick 3 choices like Rock (r), Paper (P), and Scissors (S). Yes, you can lose to the bot, and also you can win.
